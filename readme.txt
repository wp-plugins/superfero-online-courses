== Description ==

Superfero Online Courses Plugin collects the template courses from superfero.com to show up your blog page

Superfero.com is an online implementation platform that will multiply the impact of everyone working with groups of people. 

Superfero works by combining social group features with a narrow focus on solving challenges within deadlines.

The widget is available in english

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Superfero Page'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go to 'Settings' and select 'Superfero Settings'
5. Go to 'Page' and select a default page named 'Superfero Course' to change the page title

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `superfero-online-courses.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
5. Go to 'Settings' and select 'Superfero Settings'
5. Go to 'Page' and select a default page named 'Superfero Course' to change the page title

= Using FTP =

1. Download `superfero-online-courses.zip`
2. Extract the `superfero-online-courses` directory to your computer
3. Upload the `superfero-online-courses` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
5. Go to 'Settings' and select 'Superfero Settings'
5. Go to 'Page' and select a default page named 'Superfero Course' to change the page title
